# Operations

```bash
sudo supervisorctl status
sudo supervisorctl restart all
sudo systemctl reload nginx
bench --site <site> migrate
bench --site <site> clear-cache
bench build
bench --site <site> enable-scheduler
ss -ltnp | grep -E ':11000|:13000'
tail -f frappe-bench/logs/web.error.log frappe-bench/logs/worker.error.log
```

System Redis on port 6379 is not a substitute for Bench Redis. Production queue/cache instances are generated by Bench and managed by Supervisor.
